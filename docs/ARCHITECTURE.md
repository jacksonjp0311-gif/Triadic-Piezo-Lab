# Architecture

## Distribution invariant

`OPEN_LAB.html` is the complete user-facing application. Development and evidence files remain modular inside the package, but distribution remains one click.

## Visual layer

The release uses the classic cinematic laboratory: bright glass panels, chromatic physical materials, controlled bloom and a dark volumetric chamber. UI brightness and specimen brightness are independent state variables.

## Scientific layers

```text
Physical manifold
    ↓
Interface manifold
    ↓
Observer witness manifold
    ↓
Temporal memory state
    ↓
Evidence certificate
```

The live classic application includes the original 3D chamber, adaptive axial FEM, condensation, fixed-port stability bridge and 2D axisymmetric field. Later 3D, observer and 4D evidence is archived in the package and summarized inside the evidence overlay.

## Governance

- residuals propose missing information;
- each level owns its correction;
- equal-DOF or full-order evidence authorizes admission;
- rejected branches remain in the archive.
